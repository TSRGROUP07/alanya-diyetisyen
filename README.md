# Alanya Diyetisyen Web Sitesi

Alanya'da hizmet veren profesyonel bir diyetisyen için SEO odaklı, modern, mobil uyumlu ve "Biz" dilinde hazırlanmış kurumsal web sitesi projesi.

## Özellikler
- **Responsive Tasarım:** Tüm cihazlarda kusursuz görünüm.
- **SEO Uyumlu:** Semantic HTML5, hızlı yükleme, Schema.org veri yapıları.
- **Modern Arayüz (UI):** Minimalist "Glassmorphism" esintileri, özel CSS animasyonları, şık SVG ikonlar.
- **Blog & Hizmetler Sistemi:** Hedef kitle aramalarına (Long-tail keywords) göre özel oluşturulmuş iç sayfalar.

## Kullanılan Teknolojiler
- HTML5
- CSS3 (Vanilla, Custom Properties)
- JavaScript (Vanilla)

Proje [dytalanya.com](https://www.dytalanya.com) adresinde barındırılmak üzere tasarlanmıştır.
